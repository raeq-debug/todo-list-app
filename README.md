# 📝 To-Do List App

A responsive to-do list app built with HTML, CSS, and vanilla JavaScript.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Automatically saves your tasks using localStorage
- Works on desktop and mobile

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- localStorage API
